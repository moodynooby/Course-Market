## 2025-05-15 - [MUI slotProps for Adornments]
**Learning:** For Material UI (MUI) TextField components in this project, use `slotProps.input` (e.g., `slotProps.input.endAdornment`) for adding adornments. This is the modern pattern used across the codebase and aligns with the MUI version being used.
**Action:** Always check `slotProps` first when customizing standard MUI input components.
